## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` or `yarn` installed globally on your machine.

Installation:

`npm install`
or
`yarn`

To Run Test Suite:

`npm test`
or
`yarn test`

![ Tests Screenshot ](/assets/images/countdown_timer_tests.png)

To Start Server:

`npm start`
or
`yarn start`

To Visit App:

`localhost:3000`
